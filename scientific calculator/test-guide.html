1. Custom Base Logarithms

You can now use logarithms with any base using the syntax: logbase(x, base)

Examples:
  • logbase(x, 2) - Binary (base-2) logarithm
  • logbase(x, e) - Natural logarithm (same as ln(x))
  • logbase(x, 10) - Common logarithm (same as log(x))
  • logbase(x*x, 3) - Base-3 logarithm of xÂ²

2. Antilog Functions

Calculate antilog (inverse logarithm) with: antilog(x) or antilog(x, base)

Examples:
  • antilog(x) - 10^x (default base 10)
  • antilog(x, 2) - 2^x (base-2 antilog)
  • antilog(x, e) - e^x (same as exp(x))

3. Derivatives

Examples:
